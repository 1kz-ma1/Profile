DX×AIが変える、10年後の日本

分野横断型 DX×AI 体験モデル

🔗 公開URL： https://1kz-ma1.github.io/DX_AI_model/

医療DXモデルを発展させ、行政・教育・医療など複数分野を横断的に分析・可視化できる "DX×AI体験モデル" として再構築。
キャラクターを選択すると、その人物視点で現在のアナログ手続き → 電子化 → AI導入後の再設計プロセスを疑似体験できる、ゲーム性の高いインタラクティブアプリ。
各分野の課題構造を「手続きの流れ」「入力項目数」「推定時間」の観点でリアルタイム可視化。
日本のDXの本質的課題を、物語性 × データ可視化で直感的に理解できるモデル。

Python Django JavaScript DX AIモデル 行政DX 医療DX 可視化 体験型アプリ UI/UX
🎮

自動生成型RPG（AI生成RPG）

🔗 GitHub： https://github.com/1kz-ma1/RPG_AI

Python × Django で構築した AI自動生成RPG。
プレイヤーの行動に応じてシナリオ・敵キャラ・イベント画像をリアルタイム生成。
生成処理は Celery + Redis（Docker）による非同期化でバックグラウンド処理し、ストレスの少ないプレイ体験を実現。
ゲーム進行データ（スコア・撃破数・現在のシーンなど）は SQLite に保存。相棒キャラ選択、選択肢生成、画像キャッシュ機能などを実装。

Python Django Docker Celery Redis SQLite AI生成 RPG Web
