# 📖 章構成ビュー - カスタムセクション名の使い方

## 🎯 新機能：自由なセクション名

章構成ビューで「第1章」「第2章」だけでなく、**自由にセクション名を入力できるようになりました！**

### 使用例

- 💼 **資格**
- 💻 **技術**
- 🚀 **プロジェクト**
- 📚 **学習記録**
- 🎨 **作品集**
- 🏆 **受賞歴**
- 📖 **第1章** ← 従来通りも可能

---

## 📝 設定方法

### 1. 管理画面にログイン

http://localhost:8000/admin/ にアクセス

### 2. ブログ記事を編集

1. 「ブログ記事」をクリック
2. 編集したい記事を選択
3. 「**ビュー切り替え用設定**」セクションを展開

### 3. セクション名を入力

**セクション名** フィールドに好きな名前を入力：

```
資格
```

または

```
技術スタック
```

または

```
第1章: 基礎編
```

### 4. セクション内順序を設定（オプション）

同じセクション内での表示順を数字で指定：

```
セクション名: 資格
セクション内順序: 1
```

```
セクション名: 資格
セクション内順序: 2
```

### 5. 保存して確認

1. 「保存」ボタンをクリック
2. ブログページ（http://localhost:8000/blog/）にアクセス
3. 「☰ 章構成」ボタンをクリック
4. 設定したセクション名でグループ化された記事を確認

## 🔧 便利な機能

### セクション名の自動ソート

セクション名は**日本語の五十音順**で自動的にソートされます：

```
あ → か → さ → た → な → は → ま → や → ら → わ
```

英語の場合はアルファベット順：

```
A → B → C → ...
```

**「未分類」は常に最後**に表示されます。

### 従来の章番号との併用

**セクション名**を入力すると、それが優先表示されます。

- セクション名がある → セクション名を使用
- セクション名がない → 「第○章」を自動生成
- 両方ない → 「未分類」

## ⚙️ 管理画面のフィールド説明

| フィールド | 説明 | 入力例 |
|----------|------|--------|
| **セクション名** | 章構成ビューでのグループ名 | `資格`、`技術`、`第1章` |
| 章番号（旧） | セクション名未設定時の代替 | `1`、`2`、`3` |
| セクション内順序 | セクション内での表示順 | `1`、`2`、`3` |
| 分野タグ | カンマ区切りのタグ | `Python,Django,Web` |
| 関連記事 | 相関図ビュー用の関連付け | 選択リストから選択 |

## 💬 Tips

### Tip 1: 絵文字を使う

セクション名に絵文字を入れると、より視覚的に：

```
💼 ビジネススキル
💻 テクニカルスキル
🎨 クリエイティブ
```

### Tip 2: 階層的な名前

セクション名で階層を表現：

```
01. 基礎編
02. 応用編
03. 実践編
```

### Tip 3: 季節や時期で分ける

```
2024年春学期
2024年夏インターン
2024年秋プロジェクト
```
